# 🏫 托班管理系统 · 小程序页面结构

> **框架**：微信小程序
> **设计原则**：老师操作 ≤ 2步，不进入3级以上菜单

---

## 一、页面总览（共 20 个页面）

```
pages/
├── index/                   首页（对外展示，访客可见）
├── teacher/                 老师模块
│   ├── login/               老师登录
│   ├── dashboard/           工作台首页
│   ├── attendance/          签到签退
│   ├── photo/               拍照
│   ├── student-picker/      选择学生（拍照后用）
│   ├── homework/            作业管理
│   │   ├── create/          新建作业
│   │   └── detail/          作业详情
│   ├── meal/                餐食记录
│   ├── student-list/        学生列表
│   ├── student-detail/      学生档案
│   ├── growth/              成长档案
│   ├── payment/             收费管理
│   ├── notices/             通知管理
│   └── settings/            设置
├── parent/                  家长模块
│   ├── login/               家长登录（邀请码）
│   ├── dashboard/           家长首页
│   ├── growth/              孩子成长档案
│   ├── homework/            孩子作业
│   └── photos/              孩子照片
└── common/                  公共
    ├── webview/             内嵌网页（H5报告用）
    └── about/               关于我们
```

### 2.3 签到签退流程

```
┌─────────────────┐
│ attendance/     │
│ 签到页          │
│                 │
│ 操作模式切换：   │
│ [🔵签到][🟠签退]│
│                 │
│ ── 已签到 ──    │
│ 小明 08:02     │
│ 小红 08:05     │
│                 │
│ ── 未签到 ──    │
│ 小华 [点我签到] │  ← 点击签到
│ 小丽 [点我签到] │
│                 │
└─────────────────┘
        │ 长按学生 → 补签弹窗
        ▼
┌─────────────────┐
│ 补签对话框      │
│ 学生：小华      │
│ 时间：08:30 [改]│
│ 原因：早上忘记签│
│ [取消]  [确认]  │
└─────────────────┘
```

### 2.4 拍照关联流程

```
┌─────────────┐
│   photo/    │
│   拍照页    │
│   [📸 拍照] │
└──────┬──────┘
       │ 拍完
       ▼
┌────────────────┐
│ student-picker/│
│ 选择学生页     │
│                │
│ 这是谁的照片？ │
│ [单人] [多人]  │
│                │
│ ┌──┐ ┌──┐ ┌──┐│
│ │明│ │红│ │华││
│ └──┘ └──┘ └──┘│
│ ┌──┐ ┌──┐ ┌──┐│
│ │丽│ │强│ │美││
│ └──┘ └──┘ └──┘│
│     ...        │
│                │
│ [✅ 确认保存]   │
└────────────────┘
       │
       ├─ 如果选的是作业类 → 跳转到 homework/create
       ├─ 如果选的是餐食类 → 跳转到 meal/create
       └─ 如果选的是普通照片 → 保存完成，返回首页
```

### 2.5 作业三步流程

```
   ┌──────────────────┐
   │ homework/create  │
   │ 新建作业         │
   │                  │
   │ 学生：小明       │
   │ 科目：[语文][数学]│
   │                  │
   │ ── 第1步：完成 ──│
   │ [拍作业照片 1-9张]│
   │ 语音备注：[🎤]   │
   │                  │
   │ [📌 暂存待批改]   │
   └──────────────────┘
            │ 暂存后，回到首页
            │ 下次进入 homework/detail
            ▼
   ┌──────────────────┐
   │ homework/detail  │
   │ 作业详情         │
   │                  │
   │ 小明 · 6月5日    │
   │ 状态：待批改 ⏳   │
   │                  │
   │ ── 第1步 ──      │
   │ [照片] [照片]    │
   │ 备注：...        │
   │                  │
   │ ── 第2步：批改 ──│
   │ [拍批改后照片]   │
   │ 评分：⭐⭐⭐⭐    │
   │ 错题数：[3]      │
   │ 语音备注：[🎤]   │
   │                  │
   │ [✅ 保存批改]     │
   └──────────────────┘
            │ 批改后 → 状态变"已批改"
            │ 下次进入
            ▼
   ┌──────────────────┐
   │ homework/detail  │
   │                  │
   │ ── 第3步：改错 ──│
   │ [拍改错后照片]   │
   │ 语音备注：[🎤]   │
   │                  │
   │ [✅ 改错完成]     │
   └──────────────────┘
            │ 改完后 → 状态变"已完成"
            ▼
        闭环结束 ✅
```
