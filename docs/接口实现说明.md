# 接口实现说明

日期：2026-06-05

## 当前接口约定

基础路径：

```text
http://localhost:8000/api
```

统一成功格式：

```json
{
  "code": 0,
  "data": {},
  "message": "ok"
}
```

统一失败格式：

```json
{
  "code": 40001,
  "data": null,
  "message": "错误说明"
}
```

老师端登录成功后，前端保存 token，后续接口使用：

```text
Authorization: Bearer <token>
```

### GET /api/students

获取在读学生列表，老师端需要 Bearer token。

可选参数：


### POST /api/homework

创建作业记录，第一步：做完 -> 拍完成证明 -> 待批改。

请求：

```json
{
  "student_id": 1,
  "subject": "数学",
  "homework_type": "课堂作业",
  "photo_ids": [1, 2],
  "remark": "今天做完比较快",
  "homework_date": "2026-06-05"
}
```

返回：

```json
{
  "code": 0,
  "data": {
    "id": 1,
    "status": "待批改"
  },
  "message": "ok"
}
```

### PUT /api/homework/{id}/grade

批改作业，第二步：批改打分 -> 拍批改情况 -> 已批改。

请求：

```json
{
  "photo_ids": [3, 4],
  "accuracy_status": "有错已讲解",
  "error_count": 2,
  "score": 8,
  "remark": "计算题错了2道，已讲解"
}
```

返回：

```json
{
  "code": 0,
  "data": {
    "id": 1,
    "status": "已批改"
  },
  "message": "ok"
}
```

```json
  "message": "ok"
```

## 更多已实现接口

### POST /api/remarks

写老师评语，老师端需要 Bearer token。

请求：

```json
{
  "student_id": 1,
  "record_date": "2026-06-05",
  "content": "今天主动举手回答问题",
  "mood_tag": "开心"
}
```

### GET /api/remarks

获取学生评语列表，老师端需要 Bearer token。

```text
GET /api/remarks?student_id=1&page=1&page_size=20
```

### POST /api/payments

记一笔缴费记录，老师端需要 Bearer token。

请求：

```json
{
  "student_id": 1,
  "fee_type": "学费",
  "amount": 2800,
  "period_start": "2026-06-01",
  "period_end": "2026-06-30",
  "status": "已缴",
  "payment_method": "微信转账",
  "remark": "6月学费"
}
```

### GET /api/payments/summary

缴费总览，老师端需要 Bearer token。

```text
GET /api/payments/summary?month=2026-06
```

### GET /api/payments/fee-standard

收费标准公开接口，无需登录。

### GET /api/growth/overview/{student_id}

成长档案概览，老师端需要 Bearer token。

返回当月出勤天数、平均作业分、作业数量、评语数量、最新评论。

### GET /api/growth/timeline/{student_id}

成长时间线，老师端需要 Bearer token。

```text
GET /api/growth/timeline/1?days=30
```

当前时间线包含：

- 作业记录
- 老师评语
