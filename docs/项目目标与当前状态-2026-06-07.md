## 六、作业系统设计确认（2026-06-07 升级版）

> v2：固定语文/数学 + 老师学科绑定 + 操作留痕 + 新建作业联动签到

### 科目管理
- 作业科目固定为 `["语文","数学"]`
- 管理员在系统管理 → 作业配置 Tab 只能查看固定科目，不能新增或删除科目
- 作业类型仍可由管理员配置

### 老师学科绑定
- 每位老师在后台分配一个学科（管理员 = 全科，可看所有）
- 老师只看到自己学科的作业
- 新建作业时科目**锁定**为老师学科，不能修改
- 管理员新建作业时可选择科目

### 新建作业流程
```
选择学生 → 只显示今日签到学生
         → 排除本学科已交作业学生（避免重复）
         → 显示"已签到 X 人 · 已交 Y 人 · 可选 Z 人"
选择作业类型（从系统配置加载）
拍照提交
```

### 操作留痕
- 首次创建：记录 `recorded_by`（创建人）
- 批改：记录 `graded_by` + `graded_at`
- 改错：记录 `corrected_by` + `corrected_at`
- 列表/详情页展示：`批改：张老师 2026-06-07 14:30`

### 作业状态解释
| 状态 | 含义 |
|------|------|
| ⏳ 待批改 | 学生做完交上来了，你还没批 |
| ✅ 已批改 | 你批完了，也给孩子讲了，孩子拿回去改了 |
| 🎉 已完成 | 孩子改完拿回来，你检查确认没问题了 |
