# 系统管理接口对接说明

日期：2026-06-06

本文档给 Claude Code 对接 `miniprogram/pages/teacher/admin/admin` 使用。当前后端接口均挂在 `/api` 前缀下，小程序里通过 `utils/api.js` 的 `api.request({ url })` 调用时，`url` 不要重复写 `/api`。

## 通用规则

所有系统管理接口都需要老师登录后的 `Bearer token`，并且当前老师必须是管理员：

```text
Authorization: Bearer <teacher token>
```

统一返回格式：

```json
{
  "code": 0,
  "data": {},
  "message": "ok"
}
```

小程序 `api.request` 已经会自动解出 `data`，所以前端拿到的是 `data` 里的内容。

普通老师访问 `/admin/*` 应返回 `403`，错误码 `40300`，文案为“需要管理员权限”。

## 老师账号

### 列表

```js
const res = await api.request({ url: '/admin/teachers' })
this.setData({ teachers: this._enrichItems(res.teachers, 'name') })
```

返回：

```json
{
  "teachers": [
    {
      "id": 1,
      "name": "管理员",
      "phone": null,
      "role": "admin",
      "is_active": true,
      "wechat_bound": false,
      "wechat_openid": null,
      "created_at": "2026-06-06T10:00:00",
      "updated_at": "2026-06-06T10:00:00"
    }
  ]
}
```

### 新建老师

```js
await api.request({
  url: '/admin/teachers',
  method: 'POST',
  data: {
    name,
    password,
    role: formRoleIndex === 1 ? 'admin' : 'teacher'
  }
})
```

可选字段：`phone`、`wechat_openid`。

### 启用/编辑老师

```js
await api.request({
  url: `/admin/teachers/${id}`,
  method: 'PUT',
  data: { is_active: true }
})
```

也可更新：`name`、`phone`、`role`、`wechat_openid`。

### 禁用老师

```js
await api.request({
  url: `/admin/teachers/${id}`,
  method: 'DELETE'
})
```

注意：这是软禁用，不物理删除。当前登录管理员不能禁用自己。

### 重置密码

前端若已经生成新密码，可传给后端：

```js
const res = await api.request({
  url: `/admin/teachers/${id}/reset-password`,
  method: 'POST',
  data: { password: pwd }
})
```

返回：

```json
{
  "teacher_id": 2,
  "temporary_password": "abc12345"
}
```

如果前端不传 `password`，后端会自动生成临时密码并返回。

## 家长绑定

### 列表

```js
const res = await api.request({ url: '/admin/parent-bindings' })
```

后端按“家长”返回：

```json
{
  "parents": [
    {
      "id": 1,
      "name": "张妈妈",
      "relation": "妈妈",
      "phone": "13800000000",
      "invite_code": "TB1476",
      "wechat_openid": "openid_xxx",
      "bindings": [
        {
          "id": 1,
          "wechat_openid": "openid_xxx",
          "is_active": true,
          "bind_at": "2026-06-06T10:00:00",
          "last_login_at": null
        }
      ],
      "students": [
        {
          "id": 1,
          "name": "张小明",
          "grade": "一年级",
          "status": "在读",
          "is_authorized": true
        }
      ]
    }
  ]
}
```

当前系统管理页按“绑定行”展示，建议前端做扁平化：

```js
const rows = []
;(res.parents || []).forEach(parent => {
  const students = parent.students || []
  const studentName = students.map(item => item.name).join('、') || '未关联学生'
  const activeBinding = (parent.bindings || [])[0]
  rows.push({
    id: activeBinding ? activeBinding.id : `parent-${parent.id}`,
    parent_id: parent.id,
    parent_name: parent.name,
    student_name: studentName,
    relation: parent.relation,
    phone: parent.phone,
    is_active: activeBinding ? activeBinding.is_active : false,
    wechat_openid: activeBinding ? activeBinding.wechat_openid : parent.wechat_openid
  })
})
this.setData({ parentBindings: this._enrichItems(rows, 'parent_name') })
```

### 禁用某个微信绑定

```js
await api.request({
  url: `/admin/parent-bindings/${bindingId}/disable`,
  method: 'POST'
})
```

这会让该微信不能自动登录。

### 取消家长查看某个学生

如果后续页面要做“只解绑某个孩子”，用：

```js
await api.request({
  url: `/admin/parents/${parentId}/unbind-student/${studentId}`,
  method: 'POST'
})
```

## 学生退班/恢复

### 学生列表

系统管理页要同时展示在读和已退班学生，请这样调用：

```js
const res = await api.request({ url: '/students?status=' })
this.setData({ students: this._enrichItems(res.students, 'name') })
```

返回：

```json
{
  "students": [
    {
      "id": 1,
      "name": "张小明",
      "gender": "男",
      "grade": "一年级",
      "school_name": "实验小学",
      "school_class": "一班",
      "status": "在读",
      "avatar_url": null
    }
  ]
}
```

### 退班

```js
await api.request({
  url: `/admin/students/${id}/withdraw`,
  method: 'POST'
})
```

后端行为：

```text
Student.status = "已退班"
StudentParent.is_authorized = false
历史数据保留
家长端不再可见该学生
```

返回：

```json
{
  "student_id": 1,
  "status": "已退班",
  "revoked_parent_links": 1
}
```

### 恢复

```js
await api.request({
  url: `/admin/students/${id}/restore`,
  method: 'POST'
})
```

后端行为：

```text
Student.status = "在读"
Student.is_active = true
不会自动恢复家长授权
```

返回：

```json
{
  "student_id": 1,
  "status": "在读",
  "restored_parent_links": 0
}
```

## 老师微信绑定

如果老师设置页或登录后提示要做微信绑定，先走开发模式获取 openid：

```js
const session = await api.request({
  url: '/auth/wechat/session',
  method: 'POST',
  data: { code: 'dev-code-or-wx-login-code' }
})
```

开发环境 `WECHAT_MOCK_LOGIN=true` 时会返回：

```json
{
  "openid": "mock_dev-code-or-wx-login-code",
  "mock": true
}
```

再绑定到当前老师：

```js
await api.request({
  url: '/auth/teacher/bind-wechat',
  method: 'POST',
  data: { openid: session.openid }
})
```

## 前端验收点

1. 管理员登录后能看到“系统管理”入口，普通老师看不到入口。
2. 普通老师即使手动打开系统管理页，请求 `/admin/*` 也应报 `403`。
3. 老师列表能加载真实账号，新建、禁用、启用、重置密码后列表刷新。
4. 家长绑定能加载真实数据，禁用绑定后家长自动登录失败。
5. 学生退班后状态变为“已退班”，家长端学生列表不再出现该学生。
6. 恢复学生后只恢复在读状态，不自动恢复家长授权。
