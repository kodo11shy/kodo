# 🏫 托班管理系统 · 数据库设计

> **数据库**：PostgreSQL（云端）
> **编码**：UTF-8
> **时间**：所有时间字段使用 TIMESTAMP，统一存储 UTC，展示时转北京时间

---

## 一、总表关系图（文字版）

```
students ★ 核心表
  ├── 1:N → attendance_records    (签到签退)
  ├── 1:N → homework_records      (作业记录)
  ├── 1:N → photos                (通过 photo_students 中间表)
  ├── 1:N → teacher_remarks       (老师评语)
  ├── 1:N → meal_student_notes    (个别餐食记录)
  ├── 1:N → payment_records       (缴费记录)
  ├── 1:N → wecom_groups          (企微专属群)
  ├── N:M → parents               (家长，通过 student_parents)
  ├── 1:N → authorized_pickups    (接送授权人)
  └── 1:N → growth_milestones     (成长里程碑)

teachers
  └── 记录谁执行的操作（签到/拍照/写评语…）

notices
  └── 独立的通知公告

photo_students ← photos
  └── 多对多关联：一张照片可以关联多个学生
```

## 二、建表语句

### 1. 老师表（teachers）

```sql
CREATE TABLE teachers (
  id            SERIAL PRIMARY KEY,
  name          VARCHAR(50) NOT NULL,
  phone         VARCHAR(20),
  role          VARCHAR(20) DEFAULT 'teacher',   -- 'teacher' | 'admin'
  login_password VARCHAR(255) NOT NULL,           -- 登录密码（哈希存储）
  is_active     BOOLEAN DEFAULT TRUE,
  created_at    TIMESTAMP DEFAULT NOW()
);

-- 默认管理员（你）
INSERT INTO teachers (name, role, login_password)
VALUES ('管理员', 'admin', '首次启动时设置');
```

### 2. 学生表（students）

```sql
CREATE TABLE students (
  id                SERIAL PRIMARY KEY,
  name              VARCHAR(50) NOT NULL,          -- 姓名
  gender            VARCHAR(4),                    -- 男 / 女
  birth_date        DATE,                          -- 出生日期
  grade             VARCHAR(50),                   -- 年级（如"三年级"）
  school_name       VARCHAR(100),                  -- 就读学校
  school_class      VARCHAR(50),                   -- 所在班级
  school_end_time   TIME,                          -- 放学时间
  pickup_method     VARCHAR(50) DEFAULT '家长自接', -- 放学方式
  address           TEXT,                          -- 家庭地址
  enrollment_date   DATE,                          -- 入学日期
  status            VARCHAR(20) DEFAULT '在读',    -- 在读 / 退托 / 毕业
  avatar_url        VARCHAR(500),                  -- 头像照片（入学照）
  interests         TEXT,                          -- 兴趣爱好
  personality       TEXT,                          -- 性格特点
  weak_subjects     TEXT,                          -- 薄弱科目
  notes             TEXT,                          -- 备注/自由记录
  is_active         BOOLEAN DEFAULT TRUE,
  created_at        TIMESTAMP DEFAULT NOW(),
  updated_at        TIMESTAMP DEFAULT NOW()
);
```

### 3. 家长表（parents）

```sql
CREATE TABLE parents (
  id            SERIAL PRIMARY KEY,
  name          VARCHAR(50) NOT NULL,              -- 家长姓名
  relation      VARCHAR(20) NOT NULL,              -- 关系：爸爸 / 妈妈 / 爷爷 / 奶奶 …
  phone         VARCHAR(20) NOT NULL,               -- 手机号
  is_primary    BOOLEAN DEFAULT FALSE,              -- 是否为主要联系人
  is_emergency  BOOLEAN DEFAULT FALSE,              -- 是否为紧急联系人
  wechat_openid VARCHAR(100),                       -- 家长微信openid（绑定后存）
  invite_code   VARCHAR(20) UNIQUE,                 -- 邀请码（绑定用）
  created_at    TIMESTAMP DEFAULT NOW()
);
```

### 4. 学生-家长关联表（student_parents）

```sql
-- 一个学生可能有多个家长（爸爸、妈妈）
-- 一个家长也可能有多个孩子在这
CREATE TABLE student_parents (
  id            SERIAL PRIMARY KEY,
  student_id    INTEGER REFERENCES students(id) ON DELETE CASCADE,
  parent_id     INTEGER REFERENCES parents(id) ON DELETE CASCADE,
  is_authorized BOOLEAN DEFAULT TRUE,              -- 是否有权限查看
  created_at    TIMESTAMP DEFAULT NOW(),
  UNIQUE(student_id, parent_id)
);
```

```sql
-- 哪些人可以接走这个孩子
```

### 7. 签到签退表（attendance_records）

```sql
CREATE TABLE attendance_records (
  id                  SERIAL PRIMARY KEY,
  student_id          INTEGER REFERENCES students(id) ON DELETE CASCADE,
  checkin_time        TIMESTAMP NOT NULL,            -- 签到时间
  checkout_time       TIMESTAMP,                     -- 签退时间
  pickup_person       VARCHAR(50),                   -- 谁接走的
  checkin_by          INTEGER REFERENCES teachers(id), -- 谁操作的签到
  checkout_by         INTEGER REFERENCES teachers(id), -- 谁操作的签退
  is_makeup           BOOLEAN DEFAULT FALSE,          -- 是否补签
  makeup_reason       TEXT,                           -- 补签备注（必填）
  date                DATE NOT NULL,                  -- 日期（冗余，方便查）
  created_at          TIMESTAMP DEFAULT NOW(),
  UNIQUE(student_id, date)  -- 每个学生每天一条签到记录
```

### 8. 照片表（photos）

```sql
CREATE TABLE photos (
```

## 三、关键查询示例

### 学生成长时间线（汇总所有模块的数据）

```sql
-- 某学生的完整时间线（近30天）
SELECT * FROM (
  -- 作业记录
  SELECT
    hr.homework_date AS event_date,
    'homework' AS event_type,
    CONCAT(hr.subject, ' · ', hr.accuracy_status) AS title,
    hr.auto_comment AS description,
    hr.score AS score
  FROM homework_records hr
  WHERE hr.student_id = 1 AND hr.homework_date >= NOW() - INTERVAL '30 days'

  UNION ALL

  -- 老师评语
  SELECT
    tr.record_date AS event_date,
    'remark' AS event_type,
    '老师评语' AS title,
    tr.content AS description,
    NULL AS score
  FROM teacher_remarks tr
  WHERE tr.student_id = 1 AND tr.record_date >= NOW() - INTERVAL '30 days'

  UNION ALL

  -- 餐食特别记录
  SELECT
    msn.created_at::DATE AS event_date,
    'meal' AS event_type,
    '🍱 餐食记录' AS title,
    msn.remark AS description,
    NULL AS score
  FROM meal_student_notes msn
  WHERE msn.student_id = 1 AND msn.created_at >= NOW() - INTERVAL '30 days'

  UNION ALL

  -- 里程碑
  SELECT
    gm.milestone_date AS event_date,
    'milestone' AS event_type,
    gm.title,
    gm.description,
    NULL AS score
  FROM growth_milestones gm
  WHERE gm.student_id = 1 AND gm.milestone_date >= NOW() - INTERVAL '30 days'
) AS timeline
ORDER BY event_date DESC;
```

### 作业评分趋势（近4周）

```sql
SELECT
  homework_date,
  AVG(score) AS avg_score,
  COUNT(*) AS total_count,
  SUM(CASE WHEN accuracy_status = '全对' THEN 1 ELSE 0 END) AS perfect_count
FROM homework_records
WHERE student_id = 1
  AND homework_date >= NOW() - INTERVAL '4 weeks'
  AND score IS NOT NULL
GROUP BY homework_date
ORDER BY homework_date;
```

### 签到统计

```sql
-- 某月出勤统计
SELECT
  COUNT(*) AS total_days,
  SUM(CASE WHEN checkin_time IS NOT NULL THEN 1 ELSE 0 END) AS attended_days,
  SUM(CASE WHEN is_makeup = TRUE THEN 1 ELSE 0 END) AS makeup_days
FROM attendance_records
WHERE student_id = 1
  AND date >= '2026-06-01'
  AND date < '2026-07-01';
```

## 五、设计要点说明

| 要点 | 说明 |
|------|------|
| **照片不存数据库** | 数据库只存路径，实际文件存云服务器磁盘 |
| **软删除** | 大部分表用 `is_active` 标记，不真删数据 |
| **时间统一** | 存 UTC，前端展示转北京时间 |
| **补签标记** | `is_makeup + makeup_reason`，所有补签可追溯 |
| **多对多照片** | `photo_students` 中间表，一张照片关联多人 |
| **作业三步** | `homework_photos.step` 区分 done/graded/corrected |
| **健康信息** | 独立成表 `student_health`，不和基础信息混一起 |
| **邀请码** | `parents.invite_code` 唯一，家长扫码绑定 |
