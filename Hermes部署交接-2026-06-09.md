## 生产部署步骤

### 1. 服务器环境准备

需要一台云服务器（推荐配置：2核4G，Ubuntu 22.04 / CentOS 7+）：

```bash
# 安装 PostgreSQL
apt install postgresql postgresql-contrib
# 或
yum install postgresql-server postgresql-contrib

# 创建数据库和用户
sudo -u postgres psql
CREATE DATABASE tuoban;
CREATE USER tuoban_user WITH PASSWORD 'your-strong-password';
GRANT ALL PRIVILEGES ON DATABASE tuoban TO tuoban_user;
\q
```

### 2. 部署配置

生产环境配置在 `backend/deploy/` 目录下：

| 文件 | 用途 |
|------|------|
| `deploy/.env.production` | 生产环境变量配置 |
| `deploy/nginx-tuoban.conf` | Nginx 反向代理配置 |
| `deploy/tuoban-backend.service` | systemd 服务配置 |

**需要修改的内容：**

**`.env.production`** — 填入实际值：
```
DATABASE_URL=postgresql+psycopg://tuoban_user:你的密码@localhost:5432/tuoban
TOKEN_SECRET=生成一个随机密钥
WECHAT_APP_ID=你的小程序AppID
WECHAT_APP_SECRET=你的小程序AppSecret
WECHAT_MOCK_LOGIN=false
```

**`nginx-tuoban.conf`** — 替换 `your-domain.com` 为实际域名。

### 3. 部署执行

```bash
# 在服务器上
cd /home/ubuntu
git clone https://github.com/kodo11shy/kodo.git
cd kodo

# 后端
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 复制生产配置
cp deploy/.env.production .env
# 编辑 .env 填入实际值

# 启动测试
uvicorn app.main:app --host 0.0.0.0 --port 8000

# 正式部署（systemd）
sudo cp deploy/tuoban-backend.service /etc/systemd/system/
sudo systemctl enable tuoban-backend
sudo systemctl start tuoban-backend

# Nginx
sudo cp deploy/nginx-tuoban.conf /etc/nginx/sites-available/
sudo ln -s /etc/nginx/sites-available/nginx-tuoban.conf /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx

# SSL（使用 certbot）
sudo certbot --nginx -d your-domain.com
```

### 4. 小程序配置

在微信小程序管理后台：
1. **开发 → 开发设置** — 配置服务器域名（指向你的域名）
2. **版本管理** — 上传体验版/审核版

小程序 `config.js` 中的生产环境配置已预留注释，部署时取消注释并填入域名：

```js
// -------- 生产环境（云服务器） --------
// apiBase: 'https://your-domain.com/api',
// apiOrigin: 'https://your-domain.com',
```
